### ¿Que es agronature?
Agronature es una sistema de gestion **libre** diseñado para ayudar a la digitalizacion de las cooperativas agrícolas andaluzas. El software incluye los siguientes modulos de entre otros disponibles:
- Módulos para registrar la entradas de aceitunas por parcela
- Módulo de gestion de cooperativistas y lotes desde la web
- Módulo de generacion de informes de produccion y trazabilidad
- Módulo de inventariado facilmente migrable de otros ERP
